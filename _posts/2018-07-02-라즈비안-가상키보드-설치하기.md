원래는 스마트 미러를 만들기 위해서 라즈베리 파이3를 구입했었습니다만, 프로젝트는 잠정적으로 중단되고 방구석에서 썩어가는 신세가 되어버렸습니다.

자주 쓰는 녀석이 아니기 때문에 별도로 키보드를 배분하기도 그렇고[^keyboard] [Synergy](https://symless.com/synergy)를 쓰자니 v2부터는 굉장히 불안정해서 연결이 되다말다 하더군요. 그렇다고 [TeamViewer](https://www.teamviewer.com)를 쓰기는 과도한 것 같고.

[^keyboard]: 심지어 Plum84를 물려주니 전원부족으로 부팅자체가 안 됩니다 ㅎ

찾아보니 간단하게 가상 키보드를 설치할 수 있는 [방법](https://raspberrypi.stackexchange.com/questions/41150/virtual-keyboard-activation)이 있었습니다.

1\. `sudo apt-get update;sudo apt-get upgrade;`로 리포를 업데이트합니다. 필수는 아니지만 권장항목입니다.

2\. `sudo apt-get install matchbox-keyboard`를 입력하여 패키지를 설치합니다

3\. `sudo reboot` 를 입력하여 재부팅합니다.

이후 Menu >> Accessories >> Keyboard 항목을 클릭하여 가상 키보드를 사용할 수 있습니다. Keyboard 항목이 나타나지 않을 경우 Menu >> Preferences >> Main Menu Editor 에서 Keyboard 항목을 활성화할 수 있습니다.
